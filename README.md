# Q5-calculator
Ккалькулятор с графическим интерфейсом на Python (tkinter). Поддерживает множество математических функций, память, историю вычислений и переключение темы.

## Возможности

- **Основные операции**: +, -, *, /
- **Тригонометрия**: sin, cos, tan, asin, acos, atan (в радианах)
- **Логарифмы**: log (десятичный), ln (натуральный)
- **Степени и корни**: √, x², x^y
- **Комбинаторика**: факториал (n!)
- **Другие функции**: модуль (|x|), остаток (mod), случайное число (Rand)
- **Работа с дробями**: преобразование десятичных чисел в обыкновенные дроби (Frac)
- **Память**: MS, MR, M+, MC
- **История**: просмотр последних 20 операций, двойной клик для вставки результата
- **Темы**: переключение светлой/тёмной темы
- **Константы**: π, e
- **Удобный интерфейс**: изменяемый размер окна, адаптивная сетка кнопок

## Как запустить

### Вариант 1. Запуск .exe
1. Запустите Q5_calculator.exe
2. Готово,пользуйтесь.

### Вариант 2. Запуск из исходного кода (нужен Python 3)
1. Установите Python (если не установлен).
2. Скачайте файлы `calculator.py` и `Q5calc.png`.
3. Запустите в терминале:
